# 背景和挖掘目标

本实战案例的主要挖掘目标为根据豆瓣对《流浪地球》的短评数据进行文本挖掘及可视化的操作。

+ 主要有以下内容：

1. 获取豆瓣评论数据

2. 分析好评与差评的关键信息

3. 分析评论数量及评分与时间的关系

4. 分析评论者的城市分布情况
